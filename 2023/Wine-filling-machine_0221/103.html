instructions

Baijiu filling machine is a high-tech filling equipment which is controlled by PLC, photoelectric sensor and pneumatic actuator. Baijiu filling machine is specially used for filling edible liquids such as Baijiu, health wine, soy sauce, vinegar, condiments, vegetable oil, syrup, mineral water, etc. The filling measurement is accurate, without bubbles and drips. It can be used for filling 25-1000ml bottles of various shapes (including special-shaped bottles).
Features of Baijiu filling machine:
1. This machine is suitable for filling containers with different specifications, and can switch specifications within a few minutes.
2. Short loading cycle and strong production capacity.
3. There is no need to add spare parts for the specification, but only need to adjust it.
4. The Baijiu filling machine selects the filling quantity according to its own production capacity to determine the number of filling heads.
5. Touch operation color screen, which can display the production status, operation procedure, filling method, etc. The picture is intuitive, easy to operate and easy to maintain.
6. Each filling head is equipped with a bottle clamping device to ensure accurate positioning of the filling materials.
7. The filling adopts fixed flow channel parameters to control the filling time to achieve different calculations.
8. The precise filling time of pneumatic valve can be set to 0.01 seconds, which can control the measuring accuracy within 1% error range, reduce unnecessary material consumption and improve the economic benefits of users.
9. The measurement of each filling head can be adjusted separately to achieve the consistency of filling measurement.
10. The Baijiu filling machine has set the bottle feeding counting procedure, no bottle, no filling if the technology is not in place, and only when the number of bottles recorded by the counter is consistent with the set number of bottles to be filled can the filling begin.
11. The filling quantity can be adjusted initially to the required filling quantity, and the ideal filling measurement accuracy can be obtained after fine adjustment.
